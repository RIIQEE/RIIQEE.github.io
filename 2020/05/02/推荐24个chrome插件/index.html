chrome

chrome可能是程序员最喜欢用的浏览器之一了，它画面简洁，兼容性好，易于调试程序，更重要的是提供了丰富的插件。

如果你不会安装，请看

#列表插件
下图就是本人常用的一些插件，上部分是正在用的插件，是直接显示在地址栏后面；下部分是暂时不用的插件，需要用到时候，切换一下就阔以咯。

广告屏蔽神器

AdGuard 广告拦截器

下载地址：点击下载

Adblock Plus 广告拦截器

下载地址：点击下载

双管齐下，可以拦截所有浏览器上的广告，各种视频广告、弹窗内容也是可以正常拦截滴。

屏蔽百度全家桶

百度广告屏蔽
该插件可以屏蔽百度线上产品页面广告(搜索、新闻、贴吧、知道、音乐、图片、视频、文库等)。

下载地址：点击下载

屏蔽百度推广
百度搜索时，自动屏蔽搜索结果中的商业推广链接，让你直接找到你想要的结果，还你一个清爽的百度。

下载地址：点击下载

百度药丸
该插件可以不仅可以屏蔽百度广告推广，还能阻止百度追踪；专治百度莆田系、百度黄金眼、百度推广。

下载地址：点击下载

绿色搜索
屏蔽百度搜索结果列表中的广告信息, 将移除或者弱化显示。

下载地址：点击下载

学习

Google 翻译
Google 翻译，可以直接使用，不用fq哦
下载地址：点击下载

ImTranslator
翻译插件，ImTranslator 可以阅读10种语言的文字。

下载地址：点击下载

掘金
掘金插件是一款为设计师、程序员、产品经理提供每日发现优质内容的chrome插件，安装以后，每当打开浏览器，就会给你推荐一些优质的内容，包括 github 上的一些热门项目，闲时还可以学习学习，你值得拥有。

下载地址：点击下载

oscnews
oscnews 插件和掘金类似，每次打开页面时，显示开源中国软件更新资讯，内置文档导航，GitHub 趋势榜，linux命令索引，浏览历史记录和时钟页面等。特别是它的 Linux 命令搜索，我特别喜欢。

下载地址：点击下载

开发

FeHelper
Web开发者助手 FeHelper 插件提供了 JSON自动/手动格式化、JSON内容比对、代码美化与压缩、信息编解码转换、二维码生成与解码、图片Base64编解码转换、Markdown、 网页油猴、网页取色器、脑图(Xmind)等贴心工具，根据自己的需要选择安装，这对于程序员来说很有用啊。

下载地址：点击下载

Octotree
Octotree插件能够以树形展示 Github 项目代码。 安装之后，浏览Github上的项目，可看到左侧的树形结构，更方便查看代码。

下载地址：点击下载

Postman
程序员应该都知道吧 下载地址：点击下载

Postman Interceptor
Postman Interceptor，可以让 postman 中发送请求的时候使用这个网站的浏览器cookie,可添加Filter，对浏览器中的请求进行过滤

下载地址：点击下载

Vue devtools
Vue devtools 是用来调试 Vue 程序的，可以看到 Vuex 的状态值，路由信息，传的参数等等。

下载地址：点击下载

骚操作系列

谷歌访问助手
额，你懂的，只能访问google搜索，gmail等，不能访问油管等其他网址，但是对于我们来说足够啦；仅供学习交流使用哦。

下载地址：点击下载

oneTab
由于Chrome每个标签页都是一个独立进程，如果开了还多标签页的化，会耗费大量内存，让你的电脑不断变慢，此时可以把暂时不看的标签进行隐藏，可以提高chrome的性能和浏览效率。

下载地址：点击下载

Tity Bookmarks
Tity Bookmarks 是一个书签管理插件，可以搜索，没有广告，没有间谍软件

下载地址： 点击下载

Tampermonkey
Tampermonkey 可以说是一款神奇插件，简称为油猴，可以在上面安装各种脚本，如屏蔽网页广告，网盘全速下载，免费观看腾讯、优酷、爱奇艺等各大视频网站VIP电影，免费下载酷狗、腾讯等音乐网站歌曲，免费下载文库文档，领取京东、天猫购物券，购物比价等等，当然你也可以自己写脚本。

下载地址：点击下载

下载+
下载+插件是一款可以帮助你管理chrome下载信息、直观显示下载数量、自动扫描下载文件，并在下载完成后弹出提醒的插件。

下载地址：点击下载

Surfingkeys
Surfingkeys 可以让你像 vim 一样通过键盘来使用 Chrome 浏览器，比如跳转网页，上下左右滚屏

下载地址：点击下载
Stylish
可以利用 Stylish 为许多网站安装主题和皮肤，也可创建自己的主题和皮肤。

下载地址： 点击下载
